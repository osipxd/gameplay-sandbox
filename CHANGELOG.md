## [0.1.0] - 2026-04-15

Initial public release. This version includes the core top-down shooter loop with player movement, directional shooting, enemy spawning and pursuit, score tracking, health, restart flow, and a polished set of combat feedback systems such as recoil, hit pushback, screen shake, invincibility, death particles, score popups, and shot squash/color response. It also adds procedural textures for entities, variable fonts for UI and popups, hot-reloadable effect tuning, a browser build with a small static site shell, and deployment support through a Docker + Caddy image with GHCR publishing via GitHub Actions.
